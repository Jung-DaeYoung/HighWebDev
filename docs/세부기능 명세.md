# 🛠️ 세부기능 명세서

## 1. 회원 관리 기능 (Member)

| 기능 | 상세 설명 | 비고 |
| :--- | :--- | :--- |
| **회원 가입** | `POST /auth/signup`을 통해 아이디와 암호화된 비밀번호를 DB에 저장 | 중복 아이디 체크 포함 |
| **로그인** | Spring Security 폼 로그인을 통한 인증 (`POST /auth/login`) | 성공 시 세션 생성 |
| **로그아웃** | 세션 무효화 및 로그인 페이지로 이동 (`POST /auth/logout`) | |

## 2. 일정 관리 기능 (Schedule)

| 기능 | 상세 설명 | 비고 |
| :--- | :--- | :--- |
| **목록 조회** | `GET /schedules` - 본인의 일정만 조회 | 정렬 파라미터(`sort`) 지원 |
| **일정 등록** | `POST /schedules/create` - 제목, 내용, 마감시간(LocalDateTime) 저장 | |
| **상세 조회** | `GET /schedules/{id}` - 특정 일정의 상세 정보 표시 | |
| **일정 수정** | `POST /schedules/update` - 기존 일정 정보 업데이트 | |
| **일정 삭제** | `DELETE /schedules/{id}` - AJAX 요청을 통한 삭제 처리 | JavaScript `fetch` 사용 |
| **마감 계산** | 현재 시간과 비교하여 D-Day, 남은 시간(분) 등을 계산하여 뷰에 전달 | `ScheduleResponseDto`에서 처리 |

## 3. 메모 관리 기능 (Memo)

| 기능 | 상세 설명 | 비고 |
| :--- | :--- | :--- |
| **목록 조회** | 일정 목록 하단에 함께 표시 | `MemoService.findAllByUsername` |
| **메모 등록** | `POST /memos/create` - 제목, 내용 저장 | |
| **메모 수정** | `POST /memos/update` - 제목, 내용 수정 | |
| **메모 삭제** | `POST /memos/{id}/delete` - 삭제 처리 후 리다이렉트 | HTML Form 방식 |

## 4. 공통 및 기타 기능

| 기능 | 상세 설명 | 비고 |
| :--- | :--- | :--- |
| **보안 설정** | 특정 경로(`/`, `/schedules`, `/auth/**`) 외 모든 요청 인증 필요 | `SecurityConfig` |
| **글로벌 모델** | 모든 뷰에서 로그인 여부(`isLoggedIn`)와 사용자명(`username`) 접근 가능 | `GlobalModelAdvice` |
| **에러 처리** | 로그인 실패 시 에러 메시지 표시 로직 | |
| **DB 초기화** | `data.sql`을 통한 샘플 일정 데이터 자동 삽입 | |
