# 📅 마감 알림 '타임 테이블 플래너' 요구사항 명세서

## 1. 프로젝트 개요
사용자가 자신의 일정을 등록하고, 마감 기한(Deadline)에 따라 남은 시간을 실시간으로 확인하며 효율적으로 일정을 관리할 수 있는 웹 서비스입니다.

## 2. 주요 기능 (Functional Requirements)

### 2.1 일정 관리 (CRUD)
- **일정 등록**: 제목, 상세 내용, 마감 기한(날짜 및 시간)을 입력하여 새로운 일정을 생성합니다.
- **일정 목록 조회**: 등록된 모든 일정을 목록 형태로 확인합니다.
    - **정렬**: 마감 기한이 가장 가까운 순서(임박순)로 자동 정렬됩니다.
- **일정 상세 조회**: 특정 일정을 클릭하여 상세 내용을 확인합니다.
- **일정 수정**: 등록된 일정의 제목, 내용, 마감 기한을 변경합니다.
- **일정 삭제**: 완료되었거나 불필요한 일정을 삭제합니다.

### 2.2 마감 상태 시각화
- **D-Day 계산**: 현재 시간과 마감 기한을 비교하여 남은 시간을 표시합니다 (예: D-1, D-Day, 마감됨).
- **조건부 디자인**:
    - 마감 임박 (24시간 이내): 경고 색상(주황색/노란색) 적용.
    - 마감 경과: 위험 색상(빨간색) 및 '마감됨' 표시.
    - 여유 있음: 기본 색상 적용.

## 3. 비기능적 요구사항 (Non-functional Requirements)
- **사용자 경험(UX)**: 부트스트랩(Bootstrap) 레이아웃을 사용하여 모바일과 데스크톱 모두에서 보기 편한 디자인을 제공합니다.
- **데이터 보존**: 애플리케이션 실행 중에는 H2 인메모리 DB를 통해 데이터를 유지하며, `data.sql`을 통해 초기 테스트 데이터를 제공합니다.

## 4. 기술 스택 (Technical Stacks)
- **Framework**: Spring Boot 4.x
- **Language**: Java 21
- **Build Tool**: Gradle
- **Template Engine**: Mustache
- **Database**: H2 Database (In-memory)
- **ORM**: Spring Data JPA
- **Library**: Lombok (코드 간소화)
